Alexa Integration

This project added support for Alexa and Alexa Show devices to an existing web application. The Alexa skills app provides direct answers to specific questions and displays related images and illustrations when accessed from an Alexa Show device.

The Alexa support also includes the ability to connect the Alexa device to the user’s account within the web application. This connection provides an extra layer of security but also allows the app to send links for relevant content to the user’s web application account which then appear as messages in their inbox.

The web application was also enhanced so it could be used in a handsfree mode so any tablet or desktop could be paired with an Alexa device and the user could navigate, search, play videos, etc. all using voice commands with Alexa.
